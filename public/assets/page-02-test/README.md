# Page 02 Test Assets

本轮 `/test` 答题页没有新增独立图片素材，页面视觉主要由 HTML/CSS 实现，避免把文字或整张参考图做成死图。

## 当前素材策略

- `paper-texture`: 使用 CSS 的米白渐变、细网纹和轻微红金光晕模拟宣纸底。
- `corner-ornament`: 使用 CSS border 与圆角模拟剧情卡四角纹样。
- `flower-divider`: 使用 CSS 圆形朱砂花印与线性分隔符模拟卡片顶部装饰。
- `seal-choice`: 使用 CSS radial/linear-gradient、金边和内阴影模拟甲乙丙丁朱砂印章。
- `progress-thumb`: 使用 CSS 渐变圆点和装饰字形模拟进度条花形节点。

## 后续可替换素材

如果后续需要进一步提升还原度，可以在本目录补充以下轻量素材，并保持正文、选项和按钮文字继续由 HTML 渲染：

- `paper-texture.webp`: 更细腻的宣纸纹理。
- `corner-ornament.svg`: 可复用的古风角花。
- `flower-divider.svg`: 进度节点和题卡分隔花形。
- `seal-choice.svg`: 甲乙丙丁印章底纹，不包含文字。
- `progress-thumb.svg`: 金红花形进度节点。
