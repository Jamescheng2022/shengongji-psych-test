# 深宫命格：首页素材

本目录用于《深宫命格》第一页首页。

文件说明：

- home-bg.png：首页完整背景图，9:16 竖屏，包含宫廷场景、说明框、卖点区结构，不写死大段正文。
- home-title.png：主标题“深宫命格”，透明 PNG，毛笔字质感。
- home-cta-bg.png：开始测试按钮底图，不带文字。
- home-feature-panel.png：卖点卡片背景图。
- home-feature-icon-questions.png：30道题图标，金色线描风格。
- home-feature-icon-story.png：宫廷剧情包裹图标，金色线描风格。
- home-feature-icon-persona.png：人格画像解析图标，金色线描风格。
- home-ornament-divider.png：首页装饰分割线。
- home-footer-ornament.png：首页底部装饰。
- home-iphone-reference.png：iPhone 竖屏首页整页参考图，适合用于高保真首屏还原与透明点击热区。

使用原则：

1. 图片素材只负责氛围、结构、装饰。
2. 首页正文、按钮文字、卖点文字、免责声明文字尽量由代码渲染。
3. 不要把大段文字写死在背景图里。
4. 首页适配移动端 9:16 竖屏。
5. 后续页面素材也按 page-xx-name 目录分组管理。
