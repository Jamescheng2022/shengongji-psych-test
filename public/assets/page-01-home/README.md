# Page 01 Home Assets

This folder stores reusable visual assets for the mobile H5 homepage of "深宫命格".

Production rules:

1. The homepage must be rendered with layered HTML/CSS components.
2. Do not use the full reference image as a page background.
3. Buttons, labels, cards, navigation, and product copy must remain editable text.
4. Image assets are only used for atmosphere, brush-title texture, character art, icons, and decorative textures.

## Current Assets

- `bg-palace.webp`: Palace-window hero background without UI text.
- `title-shengongmingge.png`: Transparent brush-title PNG for "深宫命格".
- `heroine-card.webp`: Cropped character art for the flagship recommendation card.
- `icon-benminggongge.webp`: Decorative icon for "本命宫格".
- `icon-qingjie.webp`: Decorative icon for "情劫命书".
- `icon-liunian.webp`: Decorative icon for "流年签文".
- `icon-guiren.webp`: Decorative icon for "贵人小人局".
- `texture-paper.webp`: Paper texture used by ornate cards.
- `divider-gold.svg`: Gold divider ornament for section titles.

Legacy files prefixed with `home-` are retained for compatibility with older pages and can be removed after confirming no route imports them.
