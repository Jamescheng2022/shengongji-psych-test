# 深宫命格：首页素材

本目录用于《深宫命格》第一页首页。

## 生产使用原则

1. 首页必须由分层 HTML/CSS 组件渲染，不能用整张截图铺底。
2. 图片素材只负责氛围、装饰、复杂边框和图标。
3. 首页正文、按钮文字、卖点文字和免责声明由代码渲染，保持可编辑。
4. CTA 必须是真实可点击的 `Link` 元素，文案统一为“开始入宫”。
5. 旧整图参考 PNG 和透明热区方案已从生产资源中移除。

## 文件说明

- `home-bg.png`：宫廷场景背景，包含帘幕、窗棂、远景宫殿、说明框和功能区结构，不包含主要正文。
- `home-title-transparent.png`：主标题“深宫命格”，透明 PNG，毛笔字质感。
- `home-ornament-divider-transparent.png`：首页装饰分割线。
- `home-cta-bg-transparent.png`：开始入宫按钮底图，不带文字。
- `home-feature-panel-transparent.png`：功能卡片背景图。
- `home-feature-icon-questions-transparent.png`：30幕剧情图标，金色线描风格。
- `home-feature-icon-story-transparent.png`：宫廷剧情包裹图标，金色线描风格。
- `home-feature-icon-persona-transparent.png`：人格画像解析图标，金色线描风格。
- `home-footer-ornament-transparent.png`：首页底部装饰。
