# 深宫纪 Psych Test

一个以深宫叙事为外壳、以心理测评为内核、以中国古典审美为视觉表达的移动端 H5 产品。

## 产品定位

深宫纪不是宫斗小游戏，也不是随机命格生成器。

它的目标是：

> 让用户在高沉浸感的宫斗剧本中做选择，实际完成一套结构化心理测评，最后得到既有专业解释、又适合截图分享的人格结果。

## 三位虚拟专家分工

### 1. 中国古典审美前端设计师

负责：

- 移动端一屏体验
- 深宫、屏风、绢布、宫灯、印章、宣纸、暗金、朱红等视觉系统
- 文字阅读节奏
- 结果海报截图体验
- 不直接把参考图当背景，而是重构为可维护 UI 语言

### 2. 女性心理学家

负责：

- 测评维度设计
- 题目背后的心理构念
- 选项计分逻辑
- 结果解释边界
- 避免伪诊断、避免过度医疗化
- 保证娱乐性人格测试仍有心理可信度

### 3. 宫斗剧本女编剧

负责：

- 题目剧情化
- 情境冲突
- 女性关系、权力关系、情绪压迫、资源竞争
- 语言张力
- 让每道题既像剧情选择，又是真正的心理测评题

## MVP 结构

第一版不做 12 题。12 题太短，无法形成足够稳定的心理画像。

建议第一版：

- 30 道正式题
- 5 个维度
- 每个维度 6 道题
- 每题 4 个选项
- 约 6-8 分钟完成
- 中途按 5 个章节呈现，每章 6 题

## 核心页面

- `/` 首页

- `/result` 结果页
- `/poster` 分享海报页
