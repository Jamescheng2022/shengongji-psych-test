# 深宫命格

一个以三十幕宫廷剧情选择为核心、以六维处世方式观察为底层模型的移动端 H5 深度人格画像产品。

## 产品定位

深宫命格不是传统宫斗小游戏，也不是简单娱乐问卷，更不是专业心理诊断。

它的目标是：

> 让用户走完一段小说化、沉浸式的深宫人生，在 30 幕深宫抉择中照见自己的真实处世底色，最后得到既有代入感、又适合截图分享的命格画像。

## 正式结构

- 30 幕宫廷剧情选择
- 6 个命格维度：掌局、藏锋、人心、分寸、心潮、权欲
- 8 种深宫命格结果
- 首页、测试页、结果页、分享海报页、说明页

## 体验原则

- 30 幕连续剧情选择是产品核心资产，不压缩为短测试。
- 用户看到的是剧情抉择，不是量表问卷。
- 结果解释强调“你在 30 幕选择中最常使用的处世方式”，不做分数高低好坏判断。
- 本测试仅用于娱乐与自我观察，不作为专业心理诊断依据。
