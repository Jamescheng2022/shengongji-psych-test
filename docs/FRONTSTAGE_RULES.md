# Frontstage Rules

These rules control what users should see.

## Do not expose production roles

Do not show these labels on public pages:

- visual designer
- psychologist
- scriptwriter
- female operations
- expert review

They are internal workflow roles only.

## User-facing principle

Users should only feel:

- this test looks refined
- the questions are easy to answer
- the result feels accurate
- the poster is worth saving

## Test page rule

The test page should show only:

- progress
- scene
- choices

Do not show chapter names, psych dimensions, pressure points, or production logic on the quiz screen.

## Homepage rule

Homepage should sell the experience, not the team.

Good:

- 30-scene personality test
- choose by instinct
- receive a fate card

Bad:

- created by experts
- psychology framework
- design team labels
