# 三专家独立审查 Round 01

审查对象：深宫纪 Psych Test 第一版 H5。

## 1. 中国古典审美前端设计师

结论：当前版本有深色宫廷基调，但仍偏原型页面。它需要更强的视觉记忆点：宫门、帘影、命册、印章、纸纹和结果签。

问题：

- 首页主要靠文字解释，缺少第一眼的高级感。
- 测试页仍像普通问卷，缺少“入戏”的章节仪式。
- 结果页像报告，不像命格签。
- 分享页必须成为增长核心，不能占位。
- 音乐入口有了，但素材没有真正落位。

整改：

- 加入可维护 SVG 素材，不用版权图。
- 首页改为“命册封面 + 宫门剪影”的方向。
- 测试页加入章引与章回推进。
- 结果页先呈现命格签，再展开心理解释。
- 分享页读取真实结果，适合手机截图。

## 2. 女性心理学家

结论：五维模型和 30 题结构是对的，但心理证据链不足。用户需要看到“为什么我是这个结果”。

问题：

- 结果没有充分回扣用户选择。
- 关键选择只显示文本，没有解释心理含义。
- 五维数字容易被理解成好坏分数。
- 结果需要更多现实映射。
- 诊断边界要更自然地出现。

整改：

- 结果页增加“你为什么会得到这个结果”。
- 关键选择加入心理解释。
- 五维展示改为“倾向图”，弱化数字。
- 增加“现实中的你可能……”段落。
- 继续强调仅供娱乐和自我观察。

## 3. 宫斗剧本女编剧

结论：30 题已经有情境，但还不够像一段连续剧本。必须让用户记住人物和章节推进。

问题：

- 五章之间缺少章引。
- 人物没有稳定名字，用户不容易入戏。
- 题目间缺少前因后果。
- 部分选项像策略说明，动作性还可以加强。
- 结果命格需要“命运感”和“小传感”。

整改：

- 固定人物：贵妃沈令仪、掌事周姑姑、同屋阿檀、受宠姐姐宁枝。
- 五章加入章引。
- 结果页增加“深宫小传”。
- 海报页做成命签，而不是报告截图。

## 4. 统一修改清单

本轮必须完成：

1. 真实分享页。
2. 章引和角色设定。
3. 结果页证据链。
4. SVG 视觉素材。
5. 音乐路径统一为旧仓库文件名。

本轮暂不做：

1. 复杂分支剧情。
2. 后端用户系统。
3. AI 生成结果。
4. 诊断化心理语言。
