# 深宫命格心理测评模型设计

## 1. 产品定位

《深宫命格》不是临床诊断，也不是命运预测。它是一套以人格心理学为底座、以宫廷情境为包装的女性向自我探索测试。

用户表面上进入的是深宫剧情，实际回答的是压力、关系、边界、权力、风险与情绪调节中的偏好。目标不是制造随机结果，而是让用户觉得：结果像是从她的选择里推出来的。

## 2. 设计原则

1. **情境包装，心理计分**：题面必须是宫廷剧情，选项必须对应真实心理构念。
2. **强迫选择**：每个选项都应合理，没有明显正确答案，减少用户为了“显得更好”而作答。
3. **多维剖面**：命格不是单一最高分，而是六个维度组合后的整体画像。
4. **结果可解释**：结果页必须能说明“为什么是这个命格”，关键选择要能回溯。
5. **娱乐边界**：所有表达避免临床化，不做疾病、命运、优劣判断。

## 3. 六个底层心理维度

| 维度 ID | 宫廷表达 | 心理含义 | 高分表现 | 低分表现 |
|---|---|---|---|---|
| agency | 掌局 | 主动性、控制感、目标驱动 | 主动改变局面，争取资源 | 更愿等待、顺势或避免主导 |
| caution | 藏锋 | 谨慎、防御、风险规避 | 先保安全，观察后行动 | 更容易直接投入或冒险 |
| affiliation | 人心 | 关系投入、共情、连接需求 | 重视情义和他人感受 | 更独立、疏离、少受关系影响 |
| boundary | 分寸 | 自我边界、拒绝能力 | 能守底线，不轻易被吞没 | 容易让步、讨好或过度卷入 |
| emotionality | 心潮 | 情绪敏感、压力反应 | 对评价、关系和不确定更敏感 | 情绪外露少，更稳定或抽离 |
| power | 权欲 | 地位动机、策略交换、影响欲 | 善借势、重资源与位置 | 不迷恋控制他人或争夺中心 |

## 4. 题目结构

每题包括：

- `scene`：宫廷情境。
- `pressurePoint`：外显压力点。
- `hiddenConstruct`：真实测量构念。
- `choices`：四个强迫选择选项。
- `weights`：对六个心理维度的加减。
- `tags`：用于结果解释和关键选择回溯。

题目不应直接问“你是不是控制欲强”。应把控制、边界、共情、防御、权力动机嵌入宫廷行为选择中。

## 5. 命格判断方式

计算流程：

1. 累加 30 题原始分。
2. 将每个维度标准化为 0-100。
3. 每个命格定义一组理想人格剖面。
4. 计算用户剖面与各命格剖面的距离。
5. 距离最近的命格为主命格。
6. 前两名接近时，可在结果页显示副倾向。

这样可以避免“最高分维度 = 结果”的粗糙判断，也避免为了分散而强行分流。

## 6. 八个主命格剖面

| 命格 | 典型剖面 |
|---|---|
| 智谋贵妃 | 高掌局、高分寸、中高权欲、低心潮；理性控局，善于判断。 |
| 温仪贵人 | 高人心、高心潮、中分寸、低权欲；共情细腻，温柔但容易消耗。 |
| 冷月谋后 | 高掌局、高藏锋、高权欲、低人心；长期布局，克制冷静。 |
| 玲珑女官 | 高人心、高分寸、中高掌局；会做人，懂规则，能办事。 |
| 孤傲凤仪 | 高分寸、高掌局、低人心、中高权欲；自尊高，不愿讨好。 |
| 藏锋美人 | 高藏锋、高分寸、中高心潮控制；外柔内刚，谨慎含蓄。 |
| 笑面权臣 | 高权欲、高人心洞察、高掌局；善借势、会周旋。 |
| 清醒旁观者 | 高藏锋、高分寸、低权欲、低心潮；理性抽离，不轻易卷入。 |

## 7. 文案边界

结果页应使用“倾向、习惯、可能、容易”这类柔性表达，不使用“你就是、你一定、命中注定”等绝对判断。

推荐免责声明：

> 本测试用于娱乐与自我探索，不构成心理诊断或专业建议。
